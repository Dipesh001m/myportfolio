Real-Time System Monitoring with Flask, Docker, and AWS EKS

Overview

Developed a Python-based Flask application that monitors CPU and memory usage in real-time using the psutil library. The application is containerized with Docker, and the image is pushed to Amazon ECR. It is then deployed to Amazon EKS for efficient, scalable container orchestration and monitoring.

Technologies

Application: Python, Flask, psutil

Containerization: Docker

Image Registry: Amazon ECR

Deployment: Amazon EKS, Kubernetes

Infrastructure: Amazon EC2

Automation: Python (Boto3)

Deployment Steps

1. Build the Flask Monitoring App

  • Created a Flask app that collects system resource metrics using psutil.

2. Dockerization

  • Wrote a Dockerfile to containerize the Flask application.
  • Built and tested the Docker container locally.

3. Push Image to Amazon ECR

  • Used Boto3 to automate the ECR login and image push process.
  • Pushed the Docker image to a private ECR repository.

4. Deploy on Amazon EKS

  • Created Kubernetes deployment and service manifests.
  • Deployed the image from ECR to EKS for high availability and scaling.
